# AgentBox

Plateforme de gestion d'agents IA personnels. Créez, configurez et gérez vos assistants IA en quelques clics.

## Fonctionnalités

- **Mes Agents** : Vue d'ensemble de tous vos agents avec toggles on/off
- **Personnalité** : Nom, ton (formel/friendly/direct), secteur, instructions custom
- **Canaux** : WhatsApp, Email, Telegram, iMessage (premium) - activation par toggle
- **Compétences** : Emails, Calendrier, Prospection, CRM, Rappels - activation par toggle
- **Templates** : Modèles prêts à l'emploi par métier (Commercial, Support, Marketing...)
- **Activité** : Logs de ce que chaque agent a fait

## Stack technique

- Next.js 15
- TypeScript
- Tailwind CSS
- React (client-side proto)

## Développement

```bash
pnpm install
pnpm dev
```
